## Overview
This software is my final project for NEU480, fMRI Brain Decoding at Princeton University. The project focuses on inducing specific types of speech, which are represented by various zones in the
brain. Specifically, I use brain masks that are correlated with problem-solving, language coherence, and creativity, and create reward models for each that extrapolate to any sentence, and score
how much they would light up those brain regions. Using these reward models, I fine-tune a Llama model, and see how it performs on multiple benchmarks.
